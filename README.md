### CocoaPods

Use the following entry in your Podfile:

```rb
pod 'SuperAwesome', '~> 8.0'
```

Then run `pod update`.

In any file you'd like to use `SuperAwesome` in, don't forget to
import the framework with `import SuperAwesome`
